# Underground Warehouse Mechanics
- **Switch 3 (Left):** OFF closes (2,6) (Traps player!). ON opens (2,6).
- **Current Status:** Sw1: OFF, Sw2: ON, Sw3: OFF.
- **Action:** Turn Switch 3 ON.
- **Target:** Escape trap, then adjust Sw1 or Sw2.
- **Gate Logic:** Sw3 must be ON to enter/exit this room.
- **Inventory:** Full.
- **Hazards:** Avoid (15,4), (15,1), (12,9).
