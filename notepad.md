# Tile Mechanics
- FLOOR_UP_WALL: Blocks SOUTH movement. Allows NORTH movement. Found at Row 34 (X=6-8), Row 46, Row 48, Row 50.
- LEDGE_HOP_DOWN: Blocks NORTH movement. Found at Row 15.
- Vertical Walls: X=3, 5, 9, 11, 17, 20.

# Suicune Quest (Cianwood)
## Strategy: Western Corridor Ascent
Reach the northern plateau (14, 10) via the Western Corridor (X=0-2), which is the only land path past the Row 15 ledge.
1. Use the Row 33 gap (9, 33) to move from the eastern strips to the central corridor (X=6).
2. Use the Row 35 gap (5, 35) to move from the central corridor to the western corridor (X=2).
3. Walk North along X=2 to Row 12.
4. Walk East along Row 12 to (14, 12) and North to (14, 10).
