# Tile Mechanics (Global)
- FLOOR: Traversable.
- WALL / HEADBUTT_TREE / FRUIT_TREE / COUNTER: Impassable.
- TALL_GRASS: Traversable, triggers wild encounters.
- WATER: Impassable without Surf.
- LEDGE: One-way movement (usually South).
- WARP / DOOR / WARP_CARPET: Map transition.
- CUT_TREE: Impassable until Cut is used.
- WALL (28, 14): Impassable. Looks like a tree but is a static wall.
- TREES: Some trees trigger a 'Headbutt' prompt when interacted with from an adjacent tile.
