# Tile Mechanics
- FLOOR: Walkable.
- WALL: Impassable.
- GRASS: Encounters.
- WATER: Use SURF.
- LEDGE: One-way.
- WARP: Transitions.
- COUNTER: Face to interact.
- TREE: Use CUT.
- WHIRLPOOL: Use WHIRLPOOL.
- WATERFALL: Use WATERFALL.

# Game Mechanics
- Ghost-types: Immune to Normal/Fighting.
- Normal-types: Immune to Ghost.
